Adze

CMS made simple.

Barebones. Nothing more, nothing less

If you're looking for something to manage how your page looks, or create a theme, Adze isn't for you. Adze focuses on letting you view and edit your blog posts, rather than bloating the CMS experience.

Why Another Blogging CMS?

Adze is a simple CMS — write your posts, categorize, and view them. No fluff. It's also completely decoupled from your web page's hosting, freeing you to integrate with preexisting projects or statically hosted sites.

How It Works

  • Write a Post

    Use Markdown to write and edit your content!

  • Add Categories

    Tag your posts to keep them organized

  • Enjoy!

    Once your posts are published, access them from anywhere simply via a GET request

How do I install it?

  • Clone the repo

                $ git clone https://github.com/benjaminj6/blog-api.git

  • Install Dependencies

                # Using npm 
    $ npm i 

    # Using yarn 
    $ yarn

  • Set up environment

    This will vary depending on where you host your deployment of Adze. If you are doing a local installation, please see the development instructions

  • Deploy!

    Consult your host's documentation regarding how to deploy. Adze will run automatically via npm start, setting up the user and password that you have specified in your environment variables.
