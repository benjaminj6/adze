Blogg

CMS made simple.

Barebones. Nothing more, nothing less

If you're looking for something to manage how your page looks, or create a theme, Blogg isn't for you. Blogg focuses on letting you view and edit your blog posts, rather than bloating the CMS experience.

Why Another Blogging CMS?

Blogg is a simple CMS — write your posts, categorize, and view them. No fluff. It's also completely decoupled from your web page's hosting, freeing you to integrate with preexisting projects or statically hosted sites.

How It Works

  • Write a Post

  • Add Categories

  • Publish

  • Enjoy!

Ok, so I want to use Blogg...how do I install it?

  • 1. Clone the repo

              $ git clone https://github.com/benjaminj6/blog-api.git

  • 2. Install Dependencies

              # Using npm 
    $ npm i 

    # Using yarn 
    $ yarn

  • 3. Set up environment variables

    This will vary depending on where you host your deployment of Blogg. If you are doing a local installation, please see the development instructions

  • 4. Deploy!

    Consult your host's documentation regarding how to deploy. Blogg will run automatically via npm start, setting up the user and password that you have specified in your environment variables.
